# Django Blog API

A simple blog backend built with Django and Django REST Framework, implementing basic CRUD operations with authentication and permission control.

這份專案是一個以 Django REST Framework 為核心的後端練習專案，目標是熟悉RESTfulAPI設計、序列化、認證機制，和Django專案結構。

---

## Features

- Blog Post CRUD (Create / Read / Update / Delete)
- RESTful API design
- Django REST Framework serializers
- Authentication & permission handling
- Session-based authentication (for browser access)
- Token-based authentication (for API clients)

---

## Tech stack

- Python
- Django
- Django REST Framework
- MySQL
- Bootstrap (for basic frontend pages)
